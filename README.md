#📚 Libraria
Sistema de gerenciamento de biblioteca desenvolvido em Java por alunos do curso de Engenharia de Computação da Universidade Estadual de Feira de Santana

## Fase 1 - Diagramas, implementação e testes

Recursos:\n
👤 Contas de usuários com as informações fornecidas, incluindo convidados\n
📕 Possibilidade de criar livros, reservas, empréstimos e relatórios\n
🖥️ Armazenamento de dados em memória volátil utilizando o padrão de Data Access Object (DAO)\n
🎮 Llógica de negócios com controladores através do padrão de Separation of Concerns (SoC)\n
🧪 Testes\n

## Fase 2 - Permanência de dados

Recursos:\n
💾 Armazenamento intercambiável entre a memória volátil e o disco rígido\n
🗣️ Arquivos de teste para o armazenamento\n
